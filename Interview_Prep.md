#### Flashcard 1
Q: What is Hadoop?
A: A framework for distributed storage (HDFS) and processing (MapReduce).

#### Flashcard 2
Q: Difference between RDD and DataFrame?
A: RDD = low-level, manual optimization; DataFrame = high-level, optimized automatically.

#### Flashcard 3
Q: What is Spark?
A: A fast distributed processing engine used for big data workloads.

#### Flashcard 4
Q: What is Delta Lake?
A: A storage layer providing ACID transactions, time travel, and schema enforcement.

#### Flashcard 5
Q: What is Auto Loader?
A: Databricks feature for incremental ingestion of new files from cloud storage.

#### Flashcard 6
Q: What is CDC?
A: Change Data Capture — captures only changed rows (insert/update/delete).

#### Flashcard 7
Q: What is ETL vs ELT?
A: ETL transforms before loading; ELT loads raw data then transforms.

#### Flashcard 8
Q: What is Sqoop used for?
A: Moving data between Hadoop and relational databases.

#### Flashcard 9
Q: What is BigQuery?
A: Google’s serverless, scalable data warehouse.

#### Flashcard 10
Q: What is a Spark Cluster?
A: A master node that coordinates and worker nodes that execute tasks.

#### Flashcard 11
Q: What is DBFS?
A: Databricks File System — internal storage layer.

#### Flashcard 12
Q: What is MLflow?
A: Tool for tracking ML experiments, models, and deployments.

#### Flashcard 13
Q: What is Parquet?
A: Columnar file format optimized for analytics.

#### Flashcard 14
Q: What is a Data Warehouse?
A: System optimized for analytics and reporting using structured data.

Flashcard 15
Q: What is a Databricks Job?
A: A scheduled task that runs notebooks, scripts, or pipelines.
